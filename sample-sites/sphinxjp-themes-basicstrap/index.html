Basicstrap Sample¶

This is a demo site for Basicstrap, generated as part of Sphinx Themes Gallery.

Important

This sample documentation was generated on Oct 27, 2022, and is rebuilt weekly.

Quickstart¶

  1. Install this theme:

    $ pip install sphinxjp-themes-basicstrap

  2. Set the following in your existing Sphinx documentation’s conf.py file:

    extensions.append("sphinxjp.themes.basicstrap")
html_theme = 'basicstrap'

  3. Build your Sphinx documentation with the new theme! [1]

Documentation¶

This theme does not have any hosted-on-the-web documentation.
